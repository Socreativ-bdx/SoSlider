### Documentation ###

#### Create basic slider
```js
const myslider = new SoSlider(element, {});
```

#### Create slider with options
```js
const slider = new SoSlider(element, {
    speed: 350,
    arrows: true,
    autoplay: true,
    infinite: true
});
```

#### Slider options

| Options | Type   | Default | Description
|---------|--------|---------|---------------------
| `speed` | `int`  | `350`   | Animation Speed
|
